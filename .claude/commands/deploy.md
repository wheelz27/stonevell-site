# Deploy

Deploy the application to production on Vercel.

## Steps
1. Run `npm run build` — fix any build errors before continuing
2. Run `npm run lint` — fix any lint errors
3. Run `npm test` — all tests must pass
4. Push to `main` branch to trigger Vercel deployment
5. Monitor the Vercel dashboard for deployment status
6. Smoke test the production URL after deployment completes
