# Deploy

Deploy the Stonevell site.

## Steps
1. Check all `.html` files open correctly in a browser with no console errors
2. Verify all internal links point to existing pages
3. Confirm phone number `(775) 484-3804` is correct on every page
4. Confirm JSON-LD schema is present and accurate on every page
5. Push to `main` branch — hosting deploys automatically from there
6. Visit https://www.stonevell.com and spot-check each service page
